# catchBus

## Description
This app gives user closest bus informations. It has a apple watch apple watch version (see screenshots).

## APIs & Frameworks
- OC Transpo API
- Google Places API
- MapKit
- CoreLocation

## CocoaPods
- SwiftyJSON
